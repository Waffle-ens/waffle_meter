# 패치노트 v2.6.1 (2026-07-08)

## [신규] 버프 오버레이
- **쿨타임 중 아이콘 회색 표시** (설정 › 버프 알림에서 켜기): 버프를 거는 스킬이 쿨타임이면 오버레이 아이콘을 회색으로 흐리게 표시해, 언제 다시 쓸 수 있는지 한눈에 볼 수 있습니다. (타격 시 쿨타임이 줄어드는 특성도 실시간으로 반영됩니다)
- **"음성만" 알림 옵션**: 각 버프를 `알림끔 / 오버레이만 / 오버레이+음성`에 더해 **`음성만`**(오버레이에는 표시하지 않고 음성으로만 알림)으로도 설정할 수 있습니다.

## [수정]
- 여러 버프가 동시에 켜질 때 시작/종료 음성 알림이 겹쳐 일부가 씹히던 문제를 수정했습니다. (이제 밀린 알림도 순서대로 모두 읽어줍니다)
- 같은 버프를 여러 명이 잇따라 시전할 때 오버레이에 아이콘이 중복으로 뜨고 음성이 두 번 나오던 문제를 수정했습니다. (하나로 합치고, 나중에 시전한 쪽 지속시간으로 갱신 · 종료 음성도 갱신된 시간을 기준으로 알려줍니다)
