# 패치노트 v2.4.3 (2026-06-30)

## [보안]
- 통계 웹의 업로드·공개 설정 요청에 기기별 전자서명을 추가해, 다른 사람이 공개된 닉네임·서버만으로 내 캐릭터를 임의로 공개하거나 동의를 위조·철회하지 못하도록 보호를 강화했습니다. 업로드와 동의 철회는 종전과 똑같이 항상 동작합니다(서버 점검·오프라인 상황 포함).

## [변경]
- 캐릭터 "공개" 전환은 이제 이 기기에서 해당 캐릭터로 전투를 한 번이라도 업로드한 적이 있을 때만 가능합니다(소유 확인). 조건이 안 되는 캐릭터는 공개 토글이 비활성으로 표시되고, 위에 마우스를 올리면 안내가 나옵니다. 비공개로 되돌리기와 동의 철회에는 아무 제한이 없습니다.
