# 패치노트 v2.0.5 (2026-06-13)

## [수정]
- 일부 상황에서 클래스 아이콘이 잘못 표시되던 문제(예: 수호성이 호법성·치유성으로 표시)를 해결했습니다. 권위 있는 직업 정보가 스킬 기반 추정을 덮어쓰도록 고쳐, 한 번 잘못 잡힌 직업이 계속 고정되던 현상을 막았습니다.
- "작업표시줄/Alt+Tab 모드"와 "아이온 활성화 시 표시" 옵션이 서로 충돌해, Alt+Tab으로 창을 전환할 때 미터기가 깜빡이거나 사라지던 문제를 해결했습니다. 이제 작업표시줄 모드는 작업표시줄·Alt+Tab 노출만 제어하고, 미터기 표시/숨김은 항상 "아이온 활성화 시 표시" 설정을 따릅니다.

## [추가]
- '무스펠의 성배(보통)' 난이도의 보스(이스카리엘·칼드릭스)를 인식하도록 보스 데이터를 추가했습니다. 보통 난이도에서도 보스전이 정상적으로 집계됩니다.

## [변경]
- 업데이트 알림이 자동으로 팝업되던 동작을 제거했습니다. 새 버전이 준비되면 미터기 헤더의 업데이트 배지로 표시하며, 원할 때 눌러서 적용할 수 있습니다.
