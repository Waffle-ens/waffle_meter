# 패치노트 v2.0.7 (2026-06-14)

## [수정]
- 파티 신청·전투 기록 패널과 전투 상세 창도, 게임으로 돌아오면 미터기와 함께 항상 화면 위로 올라오도록 했습니다. 이전에는 메인 미터만 복구되고 이 창들은 게임 뒤에 가려진 채 남을 수 있었습니다.
- 헤더 버튼에 마우스를 올린 직후 Alt+Tab을 하면 미터기가 한동안 게임 뒤에 가려진 채 올라오지 않던 현상을 해결했습니다. 이제 게임에 포커스가 돌아오는 즉시 위로 올라옵니다. (트레이로 내려둔 경우에는 올라오지 않습니다.)
- 전투 상세의 '버프 가동률 / 디버프 가동률'이 실제보다 훨씬 낮게 표시되던 문제를 해결했습니다. 전투가 끝난 뒤나 기록에서 다시 볼 때도 웹 통계와 동일한 값으로 표시됩니다.

## [추가]
- 더 두꺼운 글꼴 (EX) 버전을 추가했습니다. 설정 > 글꼴에서 'NEXON Lv2 Gothic (EX)', 'Pretendard (EX)', 'Spoqa Han Sans Neo (EX)', 'Freesentation (EX)'를 선택할 수 있습니다.
