# 패치노트 v2.0.4 (2026-06-12)

## [수정]
- 멀티 모니터 사용, 모니터 분리·해상도 변경 등으로 미터기 창이 화면 밖으로 밀려 보이지 않던 문제를 해결했습니다. 시작 시와 화면 구성이 바뀔 때 항상 보이는 위치로 되돌립니다.
- "아이온 활성화 시 표시"를 켠 상태에서 게임 플레이 도중(특히 안티치트로 프로세스 조회가 막히는 환경)에 미터기가 깜빡이거나 사라지던 문제를 보강했습니다. 잠깐 다른 창으로 전환해도 곧바로 숨지 않습니다.

## [추가]
- 클릭스루(입력 통과) 모드가 켜져 있으면 헤더 왼쪽에 노란 자물쇠 아이콘을 표시해, 지금 클릭이 게임으로 통과되는 상태임을 알 수 있습니다.

## [변경]
- 미터 숫자(딜량·점유율)를 고정폭으로 표시하고 점유율을 가운데 정렬해, 값이 바뀔 때 미세하게 흔들려 보이던 정렬을 다듬었습니다.
- 설정 창의 중복된 "미터 투명도" 슬라이더를 제거했습니다. 투명도는 미터기 헤더의 슬라이더로 동일하게 조절할 수 있습니다.
