# 패치노트 v2.0.11 (2026-06-17)

## [수정]
- 게임이 활성화된 상태에서도 가끔 오버레이가 사라지던 문제를 개선했습니다. 오버레이 표시 방식을 다시 설계해, 항상 위에 떠 있는 상태를 유지하면서 숨길 때는 투명하게만 처리하도록 바꿨습니다. 게임으로 돌아왔을 때 더 안정적으로 다시 나타납니다. (전체 화면 독점 모드는 OS 특성상 제외)
- VPN·게임 부스터를 사용할 때 같은 패킷이 두 번 잡혀 DPS가 실제보다 부풀려지던 문제를 해결했습니다.
- '직업 강조' 색상 모드에서 전투가 끝나고 '대기 중' 상태가 되면 내 캐릭터 바 색이 직업색으로 풀리던 문제를 해결했습니다. 이제 전투 기록처럼 전투 종료 후에도 '내 캐릭터' 색을 유지합니다.
- 설정 버튼(또는 우클릭 메뉴)을 여러 번 누르면 설정 창이 누른 만큼 겹쳐 열리던 문제를 해결했습니다. 전투 기록·파티 신청 패널처럼, 열린 상태에서 한 번 더 누르면 창이 닫힙니다.
