# 패치노트 v2.0.8 (2026-06-15)

## [수정]
- 풀스크린 게임에서 돌아왔을 때 미터기가 게임 뒤에 가려진 채 안 보이고, 트레이 아이콘을 한 번 건드려야 다시 나타나던 문제를 근본 해결했습니다. (작업표시줄 숨김 처리가 만든 숨은 소유 창 때문에 미터기가 게임 뒤에 묶여 있던 원인을 제거했습니다.)
- 전투가 끝난 뒤 기록(히스토리)에서 저장된 전투를 다시 열어 플레이어 상세를 볼 때, 아주 가끔 누적 피해량·각종 비율·스킬 목록이 전부 0이나 빈칸으로 뜨던 문제를 해결했습니다. 이제 보고 있는 전투를 기준으로 올바르게 표시하고, 표시할 데이터가 없을 때는 '데이터 없음'으로 안내합니다.
- '직업 강조' 색상 모드에서 직업별 색을 바꾸면 내 캐릭터 바까지 직업색을 따라가던 문제를 해결했습니다. 본인으로 인식되면 '내 캐릭터' 색이 직업별 색보다 항상 우선 적용됩니다. (특히 기록 재생 때 본인 바가 직업색으로 보이던 현상)
- 같은 파티원의 직업이 실제와 다르게(예: 살성이 궁성으로) 표시되고 전투가 끝나도 그대로 유지되던 문제를 해결했습니다. 이제 그 사람이 실제로 사용한 직업 전용 스킬을 가장 우선하는 근거로 삼아 직업을 판별·교정합니다.
