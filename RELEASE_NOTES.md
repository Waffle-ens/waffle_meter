# 패치노트 v2.0.6 (2026-06-14)

## [수정]
- 방송 송출·P2P 등 다른 프로그램이 네트워크를 크게 점유하는 중에 전투해도 데미지/DPS가 정상 집계되도록 개선했습니다. 이전에는 이런 대용량 트래픽이 캡처를 포화시켜, 전투 중인데도 미터기가 "전투 대기 중"에 머무는 경우가 있었습니다.
- 미터기가 여러 개 실행되어 한쪽에 "캡처 시작 실패(캡처 헬퍼 연결 실패)"가 뜨던 문제를 해결했습니다. 트레이로 내려가 있거나 자동으로 숨겨진 상태에서 바로가기를 다시 실행하면, 새 창을 또 띄우지 않고 이미 실행 중인 미터기를 다시 보여줍니다. (이미 실행 중이면 그 사실을 바로 안내하고, 캡처 헬퍼 연결도 일시적으로 실패하면 자동으로 재시도합니다.)
- 미터기가 게임 화면 뒤로 가려져 보이지 않거나 깜빡이던 문제를 해결했습니다. (절전/대기에서 복귀할 때 항상 위 표시를 다시 잡고, "항상 표시"에서도 깜빡임 없이 고정됩니다.)
- 미터기를 오래 켜 두면 시스템 전체가 점점 느려지던 현상을 완화했습니다. (캡처 내부 버퍼가 무한정 쌓이지 않도록 상한을 두었습니다.)
- Alt+Tab으로 오래 다른 창을 보다가 게임으로 돌아왔을 때 미터기가 곧바로 다시 표시되도록 했습니다.
- 파티 신청 패널의 닫기(✕) 버튼이 확실히 닫히도록 고치고, 보강된 스킬 배지가 곧바로 표시되도록 했습니다.

## [추가]
- 미터 막대 색상 방식을 선택할 수 있습니다. '본인 강조'(내 막대만 강조) 또는 '직업 강조'(직업별로 색 지정) 중에서 고를 수 있습니다.
- 메인 미터 행에 [서버] 약칭 표기를 복원하고, 표시 여부를 켜고 끄는 토글을 추가했습니다.
- 파티 신청 패널이 자동으로 뜨지 않게 하는 토글을 추가했습니다. (자동 표시를 꺼도 헤더 버튼으로 직접 열 수 있습니다.)
- 헤더 투명도 슬라이더의 최소값을 0까지 내릴 수 있도록 했습니다. (이전 투명도 범위 복원)
