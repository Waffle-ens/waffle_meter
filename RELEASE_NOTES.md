# 패치노트 v2.6.3 (2026-07-10)

## [신규]
- 버프 오버레이 프리셋: 버프 설정을 슬롯 3개에 나눠 저장하고 바로 전환할 수 있습니다. 직업별 표시·음성 선택은 물론 아이콘 크기·글씨 색상·투명 배경·쿨타임 회색·시작/종료 음성까지 슬롯마다 통째로 담깁니다. 슬롯을 고르면 즉시 적용되고, 이후 바꾼 버프 설정은 선택된 슬롯에 자동 저장됩니다. 슬롯 이름도 수정 가능합니다. (설정 › 전투 보조 / 버프 알림)

## [수정]
- 미터를 켜면 인게임 프레임이 떨어지던 문제를 수정했습니다. 오버레이가 화면을 실제로 가리지 않는 시스템 창(1픽셀짜리 보조 창)을 "가려졌다"고 잘못 판단해, 게임이 켜져 있는 내내 초당 열 번 가까이 창 순서를 바꾸고 있었습니다. 그때마다 Windows가 게임의 화면 출력 경로를 다시 계산하느라 프레임이 흔들렸습니다. 이제 오버레이를 실제로 덮는 창이 있을 때만 순서를 되찾습니다. (증상은 그래픽 설정에 따라 나타나는 환경과 그렇지 않은 환경이 나뉩니다.)
- 미터 글씨가 살짝 번져 보이던 문제를 개선했습니다. 글자를 픽셀 격자에 맞춰 그리도록 바꿔 더 또렷해집니다.
