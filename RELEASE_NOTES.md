# 패치노트 v2.6.4 (2026-07-10)

## [변경] 전투 상세 화면 개편
- 전투 상세를 통계 사이트와 같은 형태로 정리했습니다. 위쪽 요약이 10칸으로 늘어 **DPS**와 **타수**를 함께 보여주고, 접었다 펴던 항목들은 **스킬 / 버프 업타임 / 디버프 업타임** 탭으로 바뀌었습니다.
- 스킬 표에 **DPS · 평균 피해 · 비중** 항목이 생겼습니다. 지속피해(도트)는 이름 뒤 표기 대신 **직접 / 지속** 배지로 구분하고, 행 뒤에는 피해 비중만큼 음영이 깔려 순위가 한눈에 들어옵니다.
- 버프 업타임은 이제 **본인이 사용한 버프만** 보여줍니다(파티원이 걸어준 버프 제외). 디버프 업타임도 **본인이 보스에게 건 디버프만** 표시합니다.
- 파티원 목록은 미터 본체에서 이미 보이므로 상세 화면에서 뺐습니다.

## [수정]
- 전투 상세의 버프/디버프 가동률에서 **같은 스킬이 여러 줄로 나뉘어 보이던 문제**를 수정했습니다. 스킬 하나는 실제로 여러 개의 내부 효과(랭크·단계·부가 효과)를 동시에 거는데, 미터가 그걸 각각 다른 버프로 세고 있었습니다. 검성 [살기 파열]은 버프 가동률에 두 줄, [격노 폭발]은 보스 디버프에 세 줄, 정령성 [정령 강림]과 수호성 [격앙]은 두 줄로 나뉘어 있었습니다. 이제 스킬 하나당 한 줄로 합치고, 가동률은 그 스킬이 **실제로 걸려 있던 시간 전체**를 기준으로 계산합니다(겹치는 구간을 두 번 세지 않습니다).
- 이름만 같고 실제로는 다른 스킬은 **계속 따로** 표시됩니다. 예를 들어 정령성 [지연 피해]는 서로 다른 두 스킬이라 두 줄로 남습니다.
- 한 스킬이 **나에게 거는 버프**와 **보스에게 거는 디버프**를 정확히 나눠 표시합니다. 예를 들어 검성 [격노 폭발]은 버프 가동률에 한 줄(자신의 피해 증폭이 오르는 효과), 보스 디버프 가동률에 한 줄(대상의 공격력이 깎이는 효과)로 각각 잡힙니다.
- 일부 **자가 버프가 "그 외"로 잘못 분류**되던 문제를 수정했습니다. 데이터에 이름이 등록되지 않은 랭크의 버프가 직업 판별에 실패해, 주문서·물약 같은 소모품과 한 칸에 묶이고 있었습니다. 이제 내 직업 버프는 "내 버프"에 정확히 들어갑니다.

## [변경]
- 통계 사이트로 보내는 전투 데이터에 스킬 기준 코드를 함께 담아, 사이트에서도 미터와 같은 기준으로 버프를 묶을 수 있게 했습니다. (사이트 화면 반영은 별도 배포)
