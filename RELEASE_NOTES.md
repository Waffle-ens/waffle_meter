# 패치노트 v2.8.0 (2026-07-22)

## [수정] 전투 중 파티원이 안 보이던 문제
- 전투 중 **파티원이 한 명씩 사라지던** 문제를 수정했습니다. 이름을 아직 받지 못한 참가자의 행이 통째로 숨겨지는 것이 원인이었고, 딜은 계속 합계에 들어가 있어 **보이는 비중의 합이 100%가 되지 않는** 증상으로 나타났습니다. 이제 직업으로 누구인지 확실히 가려지면 이름을 채워 표시합니다.
- 이번 접속에서 **한 번도 마주친 적 없는 파티원**도 파티 목록을 근거로 구제해 표시합니다.
- **낯선 사람이나 소환수 행이 끼어들어 실제 파티원이 잘려 나가던** 문제를 수정했습니다. 확정된 파티원과 본인은 표시 인원 제한에서 제외됩니다.
- 다른 캐릭터가 **이전 캐릭터의 이름으로 표시**되어, 정작 그 파티원은 사라진 것처럼 보이던 문제를 수정했습니다.
- 저장된 전투를 **기록에서 다시 열면 참가자가 줄어들던** 문제를 수정했습니다(예: 5명 → 4명). 전투가 끝날 때 그 전투의 파티 정보를 함께 저장하는 방식이라 **이번 업데이트 이후 새로 기록되는 전투부터 적용**되고, 이미 저장된 기록은 종전과 동일하게 표시됩니다.

## [수정] 본인 인식
- **존 이동·던전 재입장·전투 중 난입** 뒤 본인이 다시 인식되지 않아 **본인 딜이 통째로 빠지던**(본인 행·본인 색이 없고 통계에도 올라가지 않던) 문제를 수정했습니다.
- 이름과 서버가 **정확히 일치하고 실제로 그 캐릭터가 전투에 나타났을 때만** 본인으로 다시 연결합니다. 다른 서버의 동명이인은 연결되지 않습니다.

## [수정] 전투가 시작되지 않던 문제
- 미터가 **모르는 몹**이면 전투가 아예 시작되지 않았습니다. 몹 목록을 최신 데이터로 채워(6,445종 → 8,091종, 보스 267종 추가) 그동안 인식되지 않던 보스에서도 전투가 정상적으로 시작됩니다.
- **보스 등장 정보가 늦게 도착**해 전투가 열리지 않던 경우, 이제 뒤늦게라도 전투를 열고 **그 사이에 넣은 딜까지 포함**해 집계합니다(보스 한정, 다른 전투가 진행 중이 아닐 때).

## [수정] 버프 오버레이 정확도
- **일부 버프가 오버레이에 아예 뜨지 않거나**, 실제로 존재하지 않는 버프가 표시되던 문제를 수정했습니다.
- 버프가 **예정보다 일찍 풀렸는데도 남은 시간이 다 흐를 때까지 계속 표시**되던 문제를 수정했습니다. 이제 풀리는 즉시 사라집니다.
- **사망하면 버프 표시를 초기화**합니다. 부활 시점에는 실제로 모든 버프가 풀린 상태이기 때문이며, 이때는 버프 종료 음성이 나오지 않습니다.
- 치유성 **'대지의 축복'**이 '대지의 징벌' 이름·아이콘으로 표시되던 문제를 수정했습니다. 이제 인게임과 같은 이름·아이콘으로 따로 표시됩니다.
- **함께 적용되지 않는 버프 쌍**(노련한 반격↔격앙, 보호의 빛↔불패의 진언, 대지의 축복↔질풍의 권능)은 실제로 효과가 살아 있는 쪽만 표시합니다.

## [수정] 버프 오버레이 표시
- **폭주**처럼 계속 유지되는 버프가 **유지 중인데도 오버레이에서 사라지고 "오프" 음성이 나오던** 문제를 수정했습니다. 끝나는 시각이 정해져 있지 않은 버프라 **남은 시간 링과 숫자는 표시하지 않습니다.**
- '**미터를 숨겨도 오버레이 유지**'를 켠 상태로 미터를 숨겼을 때 버프 오버레이가 **깜빡이던** 문제를 수정했습니다.

## [수정] 대상 HP 표시
- 교전이 시작되는 순간 보스의 **남은 HP가 잠깐 최대치로 튀어 보이던** 문제를 수정했습니다.
- 서버가 최대 HP를 알려주는 경우 그 값을 그대로 사용해 **HP 비율 표시가 더 정확**해집니다.

## [수정] 파티 참가 요청 패널
- 같은 사람이 **다시 신청하면 스킬 뱃지가 비어 보이던** 문제를 수정했습니다. 이미 조회해 둔 뱃지를 그대로 이어서 보여줍니다.

## [추가] 버프 오버레이 표시 순서 (설정 › 버프 알림)
- 오버레이에 버프를 늘어놓는 기준을 **적용 순서(기본) / 남은 시간 순 / 이름 순** 중에서 고를 수 있습니다. 기본값인 '적용 순서'는 버프가 유지되는 동안 자리가 바뀌지 않아 가장 안정적이고, '남은 시간 순'은 타이머가 흐를 때마다 아이콘이 움직입니다.
- 버프 목록에서 **'위치 고정'**을 켠 버프는 정렬 기준과 무관하게 **항상 버프 오버레이 맨 앞**에 표시됩니다.
- 지금까지는 정렬이 실제로 반영되지 않고 **처음 뜬 순서 그대로 고정**돼 있었는데, 이제 고른 기준대로 정렬됩니다. 표시 순서와 고정 목록은 **버프 프리셋 3슬롯**에도 함께 저장됩니다.

- 본인 위치는 패킷 제약상 **스킬 시전 시점에만 정확**합니다. 위치를 알 수 없는 구간은 이제 **속 빈 링(마지막 위치)**으로 표시해, 실제로 그 자리에 멈춰 있던 것으로 오해하지 않도록 했습니다.

## [변경] 리플레이 저장 관리
- 리플레이 저장 파일을 **최근 50개까지만** 보관하고 오래된 파일은 자동으로 정리합니다.


# 패치노트 v2.7.2 (2026-07-16)

## [수정] 전투 리플레이 — 장판 표시 정확도
- 일부 보스 장판이 **실제보다 훨씬 넓게(도넛 전체처럼)** 그려지던 문제를 수정했습니다. 이제 부채꼴/고리형 장판이 실제 각도대로 표시됩니다.
- **장판 공격이 아닌 스킬**(보스의 버프·회복 오라, 넓은 범위 표식 등)이 장판처럼 표시되던 문제를 수정했습니다. 플레이어에게 피해를 주는 실제 장판만 표시됩니다.

## [수정]
- 특정 던전에서 첫 보스가 **드물게 인식되지 않을 때 미터가 종료되던 문제**를 수정했습니다. 이제 그런 경우에도 종료 없이 "타겟 인식 실패"로 표시됩니다.
