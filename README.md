# waffle_meter

아이온2 전투분석을 위한 미터기 프로젝트

해당 프로젝트는 아이온2 운영측의 요청, 패킷암호화등의 조치, 공식적인 사용중단 언급이 있다면 중지 및 비공개상태로 전환됩니다.

## 사용법

1. [Npcap](https://npcap.com/#download)을 설치합니다.
   설치 중 `Install Npcap in WinPcap API-compatible Mode` 옵션을 반드시 체크합니다.

2. [Releases](https://github.com/Waffle-ens/waffle_meter/releases)에서 최신 `waffle_meter-x.x.x.msi` 파일을 다운로드해 설치합니다.

3. 아이온2가 실행 중이라면 먼저 캐릭터 선택창으로 이동합니다.

4. 시작 메뉴 또는 설치 폴더의 `waffle_meter.exe`를 관리자 권한으로 실행합니다.
   기본 설치 경로는 사용자별 설치 경로인 `%LOCALAPPDATA%\waffle_meter`입니다.

5. 미터기가 보이면 아이온2에 접속해 전투를 시작합니다.
   데미지 기록, 전투 기록, 파티 신청 패널은 인게임 오버레이 형태로 표시됩니다.

6. 미터기가 보이지 않거나 위치가 이상하면 앱을 종료한 뒤 `%APPDATA%\waffle_meter.v1.6\settings.properties`에서 `windowX`, `windowY`, `uiX`, `uiY` 값을 `0`으로 수정한 다음 다시 실행합니다.

7. 업데이트 알림이 표시되면 앱 안에서 업데이트 파일을 다운로드하거나, [Releases](https://github.com/Waffle-ens/waffle_meter/releases)에서 최신 MSI를 직접 설치합니다.

## 업데이트 기록

- v1.7.5
  - 미터기 바깥의 보이지 않는 그림자 영역까지 클릭이 잡히던 문제를 수정했습니다.
  - 미터기 드래그, 리사이즈, 버튼 클릭 반응이 느려질 수 있던 문제를 개선했습니다.
  - 아이온2 전체화면 위에서도 오버레이가 더 부드럽게 움직이도록 렌더링 방식을 조정했습니다.
  - 설정에서 실제 사용성이 떨어지던 GPU 가속, 렌더링 프레임 제한 항목을 제거했습니다.

- v1.7.4
  - 본인 전투력이 변경될 때 패킷에서 직접 갱신되도록 보강했습니다.
  - 오버레이가 아이온2 포커스를 빼앗아 게임 FPS가 떨어질 수 있는 문제를 줄였습니다.
  - 미터기 테두리 밖 보이지 않는 드래그/리사이즈 영역을 줄여 화면 끝 배치가 더 정확해졌습니다.
  - 설정에 오버레이 렌더링 프레임 제한을 추가해 GPU 부하를 조절할 수 있도록 했습니다.

- v1.7.3
  - 통계 웹으로 전송하는 전투 기록 형식을 보강했습니다.
  - 스킬 피해량을 직접 피해와 지속 피해로 나누어 전송해, 통계 웹에서 DoT 지분을 더 정확히 집계할 수 있도록 했습니다.
  - 버프/디버프 가동률에 시전자 식별 정보를 추가해, 같은 직업이 여러 명인 파티/공대에서도 누가 건 효과인지 구분할 수 있도록 했습니다.

- v1.7.2
  - 설치 파일명이 `waffle_meter-버전.msi` 형식으로 정리되어 버전이 중복 표시되던 문제를 수정했습니다.
  - 업데이트 파일을 백그라운드로 내려받고, 다운로드 완료 후 즉시 재시작하여 적용하거나 앱 종료 시 자동 적용할 수 있도록 개선했습니다.
  - 사용자별 설치 방식으로 전환해 이후 업데이트가 기존 설치 위에 더 안정적으로 적용되도록 했습니다.
  - 바탕화면 바로가기는 설치 프로그램의 기본 생성 방식을 사용하도록 정리했습니다.
  - 이번 전환 버전은 기존 `waffle_meter.v1.6` 설치와 별도 앱으로 설치될 수 있어, 기존 버전이 남아 있다면 한 번 제거한 뒤 설치하는 것을 권장합니다.

- v1.7.1
  - 본인과 파티원의 전투력 표시가 더 안정적으로 잡히도록 캐릭터 정보 패킷 파싱을 보강했습니다.
  - 본인 전투력은 캐릭터 감지 시점에 공식 캐릭터 정보를 함께 조회해 보완하도록 했습니다.
  - 통계 웹 동의/업로드가 최신 서버 주소와 정상 동기화되도록 수정했습니다.
  - 캐릭터가 아직 감지되지 않은 상태에서는 통계 동의/업로드 설정을 변경할 수 없도록 안내를 추가했습니다.

- v1.7.0
  - 바탕화면 바로가기 생성 과정을 보강했습니다.

- v1.6.9
  - 통계 웹 연동을 정식 반영했습니다. 동의한 캐릭터의 전투 기록은 보스 처치 전투에 한해 통계 웹으로 전송됩니다.
  - 통계 웹에서 버프/디버프 가동률이 더 정확히 구분되도록 전송 정보를 보강했습니다.

  - 파티 신청창에서 선택한 일반 스킬/스티그마가 신청 카드에 보이지 않던 문제를 수정했습니다.

  - 닉네임 옆 전투력 배지를 정리해 같은 전투력이 중복 표시되지 않도록 했습니다.
  - 상세내역의 스킬 아이콘, 버프/디버프 아이콘, 툴팁 표시를 보강했습니다.
  - 전투 기록은 최근 12개까지 보관하도록 늘리고, 보스 전투 요약 저장 방식을 개선했습니다.
  - 전투 종료 후 같은 보스 기록이 여러 번 쌓이거나 시간이 계속 늘어나는 문제를 줄였습니다.
  - 메인 미터기에 전투 정보 초기화 버튼을 추가하고, 설정에서 초기화 단축키를 지정할 수 있도록 했습니다.
  - 설정창을 표시, 오버레이, 테마, 통계, 초기화 탭으로 정리했습니다.
  - GPU 가속 설정, 창 크기 조절, 다중 모니터 이동, 화면 가장자리 배치 동작을 보강했습니다.
  - 알트탭 이후 작업표시줄이 가려지거나 오버레이가 클릭되지 않는 문제를 줄이도록 오버레이 동작을 개선했습니다.
  - 종료 버튼을 눌렀을 때 트레이로 보낼지 완전히 종료할지 선택할 수 있도록 했습니다.
  - 업데이트 알림을 팝업 대신 우측 하단 토스트 알림으로 표시하도록 변경했습니다.
  - 신규 성역 `무스펠의 성배`와 `붉은 연심의 거울` 단계별 보스 인식 정보를 보강했습니다.
